Getting support

Please prepare yourself for support as follows (for munshi9 users only):
  1. Install teamviewer and set its password to 'hello123' and keep it running always. Click here to download the teamviewer.
  2. Install winscp. Click here to download winscp.
  3. Install putty. Click there to download putty.
  4. Connect to your server with putty and user root.
Important:Install teamviewer version from the link above. Any other version may not work.

Email is our preferred way to communicate for all kind of issues.
