Editing templates

All document printing (invoices, orders, regcard, reservation) is defined as templates. A template is an html file. You just need some basic html knowledge to edit a template.

There are two type of html templates; HTML and TT. Both are same from user point of view; the minor technical details don't matter for minor editing.

To edit templates:

  1. Goto System--html templates or System--TT templates
  2. Click on the template you want to edit. It will open in a window and you can make changes.
