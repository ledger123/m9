Software needed

  1. putty: Putty is used to connect to munshi9 server command line. Click here to download and run.
  2. winscp: Winscp is used to copy files from or to munshi9 server. Click here to download and install

Full upgrade

  1. Login with putty directly on server. To login type following:
    2. root@server-ip-address
    and when asked type the server password for root.
  2. Run following commands.
    3. cd /root/munshi9/
./download.sh

Full upgrade without Internet on Server

  1. Download munshi9.sql.zip
  2. Copy munshi9.sql.zip in /root/munshi9/ using winscp.
  3. Login to server with putty and run following commands: 
    cd /root/munshi9/
unzip munshi9.sql.zip
sqlplus munshi8/gbaba4000 @munshi9.sql

Minor update

  1. Login with putty or use terminal on server.
  2. Run following commands.
    3. cd /root/munshi9/
sqlplus munshi8/gbaba4000
   ... Now 'SQL>' will appear. Here you will type the name of the update script as follows
@http://mavsol.com/munshi9/filename

Minor update without Internet on server

  1. Download http://mavsol.com/munshi9/filename.sql
  2. Copy filename.sql to /root/munshi9/ folder using WINSCP
  3. Login with putty or use terminal on server.
  4. Run following commands.
    5. cd /root/munshi9/
sqlplus munshi8/gbaba4000
   ... Now 'SQL>' will appear. Here you will type the name of the update script as follows
@filename.sql

github update

  1. Login with putty using root@ip-address and run the following commands. 
    cd /root/munshi9/
git pull
  2. Let us know if you get any error with this command.

Running an SQL query

  1. Logon to SQL*Plus as shown above in minor update.
  2. You should be now on SQL> prompt.
  3. Here type the query we have sent you in email. It is better to copy/paste the query to avoid any typing error.
  4. Carefully see the message after query (rows updated, rows deleted or ANY ERROR) and report us this message in reply.
