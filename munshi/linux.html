Important Linux knowledge

File system

  1. Terminal window is opened with Alt-Ctrl+T on server.
  2. putty can be used to connect to terminal from windows computer.
  3. 'nano' is a simple text editor which can be used from putty or terminal.
  4. 'ls -l' is used to show the files in the folder.
  5. 'cd foldername' is used to move to subfolder.
  6. 'cd /folder1/folder2/' is used to move to any folder.
  7. 'cp /folder1/file1.txt ./' is used to copy 'file1.txt' to current folder.
  8. 'mv /folder1/file1.txt ./' is used to move file from a folder to current folder.
  9. Desktop folder for a user is '/home/username/Desktop/'.

Networking

  1. 'ifconfig' is used to see the ip numbers of the server from command line.
  2. 'ip route list' is used to see the default gateway from command line.
  3. ping command is used to test network connectivity.
  4. 'ping oracle.com' will show an ip number if the internet is working on the server.

Super user

  1. 'root' is super user in Linux.
  2. You can login to root user with 'su -' command and specifying the root password.
  3. You need to set/change the root password first time with 'sudo su passwd'.
  4. You can run any command as root using 'sudo command-name'.

Software installation

  1. Any software package can be installed with 'sudo apt-get install package-name'
  2. It is recommended to update definitions with 'sudo apt-get update' before installing any package on new system.
  3. It is also recommend to change the installation server to 'Main Server' using 'Edit--Software sources' in Ubuntu Software Center'.
